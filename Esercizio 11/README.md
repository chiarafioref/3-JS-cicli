# ESERCIZIO 

L'utente ha 3 tentativi per indovinare la password. Stampa "Benvenuto" se riesce nei tentativi consentiti, altrimenti "Bloccato"

## RAGIONAMENTO

- Creo una password corretta per accedere al sito
- Chiedo all'utente di inserire la password
  - SE l'utente indovina la password entro 3 tentativi
    - stampo "Benvenuto"
  - ALTRIMENTI 
    - stampo "Bloccato"
